# Greenden-clone
Greenden | Responsive Plant Shop
A modern, multi-page website for a plant and floral boutique. This project demonstrates high-quality UI/UX design, responsive layouts, and real-time search functionality.

🚀 Key Features
Multi-Page Navigation: Separate, functional pages for Home, Products, and Contact.

Live Search Filter: A JavaScript-powered search bar on the Products page that filters items instantly as you type.

Mobile-First Design: Fully responsive grid system that adapts from 1 column (mobile) to 4 columns (desktop).

Interactive Sidebar: A custom-coded mobile menu that slides in/out for a premium user experience.

Modern UI Components: Includes "Best Seller" grids, hover animations, and glassmorphism effects.

🛠️ Tech Stack
HTML5: Structural layout and SEO-friendly semantic tags.

Tailwind CSS: Utility-first styling for rapid, responsive design.

Vanilla JavaScript: Logic for the search engine and mobile menu toggle.

Font Awesome: Professional iconography.
